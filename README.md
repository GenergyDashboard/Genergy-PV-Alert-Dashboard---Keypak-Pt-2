# Keypak Part 2 — PV Monitoring Dashboard

Live solar PV monitoring dashboard for the Keypak Part 2 site.

## Setup

1. Create empty GitHub repo, push via git (to include `.github/workflows/`)
2. Add secrets: `FUSIONSOLAR_USERNAME`, `FUSIONSOLAR_PASSWORD`, `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`
3. Enable GitHub Pages: Settings → Pages → Source: `main`, root
